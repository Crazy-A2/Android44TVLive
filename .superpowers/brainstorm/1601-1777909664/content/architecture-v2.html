整体架构设计（修订版）

基于开机即播 + 中老年友好的交互模型

应用结构：全屏播放 + OSD 覆盖

LivePlayerActivity（唯一 Activity）
IJKPlayer 全屏播放
SurfaceView 渲染
OSD 覆盖层管理器
频道信息 / 列表 / 设置
SourceUpdateService
启动检查 + 手动触发
抓取 → 解析 → 比对 → 入库
SourceHealthChecker
播放失败 → 标记失效
失效源 → 删除 → 拉取新源
SourceRepository
SQLite 数据层
频道 + 多源 + 健康 + 收藏
SQLite (Room 1.x)
channels / sources / favorites / history
SharedPreferences
上次频道 / 音量 / 播放设置

源健康检测与自动修复流程

播放失败 → 切换备用源 → 标记源失效 → 删除失效源 → 触发源更新 → 新源入库
所有源都失效时：OSD 提示"频道暂无可用源，正在更新…"，后台立即拉取新源

遥控器按键映射

⬆ / ⬇上一个 / 下一个频道
⬅ / ➡减小 / 增大音量
OK / 确认打开频道列表覆盖层
菜单 / Menu打开设置覆盖层
返回 / Back关闭当前覆盖层
数字键直接输入频道号换台
收藏长按 OK 添加/取消收藏
静音切换静音
